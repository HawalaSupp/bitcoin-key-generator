FAILURE RECOVERY
================================================================================

If a test FAILS:

1. Identify which chain failed
   Example: "Bitcoin address mismatch"

2. Check the troubleshooting section in VALIDATION_TESTING.md
   Example: VALIDATION_TESTING.md â†’ "Troubleshooting" â†’ "Bitcoin/Litecoin"

3. Review the implementation in rust-app/src/main.rs
   Example: generate_bitcoin_keys() function

4. Fix the issue and rebuild:
   cargo build --manifest-path rust-app/Cargo.toml

5. Re-run the failed test

6. Repeat until all tests pass

Most common issues:
- Bitcoin/Litecoin: Hash160 or Bech32 encoding
- Monero: Ed25519 scalar reduction or Keccak-256
- Solana: Base58 encoding or Ed25519
- Ethereum: Keccak-256 hash or EIP-55 checksum
